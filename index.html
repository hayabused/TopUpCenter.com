Cara Pembayaran

  1. Pilih game dan isi data top up dengan lengkap pada formulir di atas.
  2. Klik tombol "Kirim Pesanan" untuk menghubungi admin via WhatsApp dengan data pesanan Anda.
  3. Admin akan menghubungi Anda untuk konfirmasi metode pembayaran dan total biaya.
  4. Lakukan pembayaran sesuai instruksi dari admin.
  5. Setelah pembayaran terkonfirmasi, admin akan memproses pengisian top up ke akun game Anda.
  6. Pengiriman top up akan dikonfirmasi melalui chat WhatsApp.

Pertanyaan yang Sering Diajukan (FAQ)

Bagaimana cara saya tahu top up sudah berhasil?

Admin akan mengonfirmasi pengisian top up melalui chat WhatsApp dan Anda akan menerima notifikasi di dalam game.

Berapa lama proses top up?

Proses biasanya memakan waktu 10-30 menit setelah pembayaran terkonfirmasi.

Apakah data saya aman?

Data Anda hanya digunakan untuk proses top up dan tidak akan dibagikan ke pihak lain.

Metode pembayaran apa saja yang tersedia?

Kami menerima pembayaran via transfer bank, e-wallet, dan pulsa, semua diproses melalui admin.
